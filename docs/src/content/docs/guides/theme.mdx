TRNovel 的界面配色由一组主题色驱动,可在「主题设置」页自定义。主页选择「主题设置」进入。

![主题设置](../../../assets/guides/theme.gif)

## 设置主题色

共有六项可配置颜色:

| 颜色项 | 用途 |
|--------|------|
| Text Color | 正文 / 普通文字 |
| Primary Color | 主色(高亮、边框标题) |
| Warning Color | 警告 |
| Error Color | 错误 |
| Success Color | 成功 / 选中 |
| Info Color | 次要信息 |

<Steps>

1. 用 `↑/↓` 选择要修改的颜色项,回车进入取色弹窗。

2. 在弹窗里:用 `↑/↓` 在预设颜色中选择回车确认,或按 `s` 直接输入颜色名(如 `Red`)或 HEX 值(如 `#FF0000`)。

</Steps>

配置即时生效并保存到 `~/.novel/theme.json`。

## 恢复默认

在主题设置页按 `d` 弹出确认框即可恢复默认配色:`y` 确认,`n` / `Esc` 取消,或用 `←/→` 选择「确认 / 取消」后回车。
