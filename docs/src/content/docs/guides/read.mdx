## 选择章节

进入阅读界面后，左侧会显示章节列表，使用上下方向键选择您想阅读的章节，然后按回车键确认。
![select-chapter](../../../assets/guides/select-chapter.png)

## 阅读

进入阅读模式后
![novel](../../../assets/guides/novel.png)
使用上下方向键翻页，按`Tab`键退出阅读模式，返回章节列表。

:::tip[技巧]
可以使用快捷键`I`查看当前页面支持的快捷键列表。
![shortcuts](../../../assets/guides/shortcuts.png)
:::

## 退出

在任何界面下，按`Q`键或者`Ctrl + C`即可退出 TRNovel 应用。
