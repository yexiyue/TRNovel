本文介绍如何用 TRNovel 查找并打开本地 `.txt` 小说。

## 启动与主页

打开终端,运行 `trnovel`(包管理器安装时也可用简写 `trn`)即可进入主页。主页有四个入口:**本地小说、网络小说、历史记录、主题设置**,用 `↑/↓`(或 `j/k`)选择,回车进入。

![主页](../../../assets/guides/home.gif)

<Aside type="tip">主页任意位置按 `i` 都能呼出当前页面的快捷键面板,不用记键位。</Aside>

## 打开本地小说

选择「本地小说」后,TRNovel 会扫描**当前所在目录**及其子目录下的所有 `.txt` 文件,以文件树呈现:目录可展开/折叠,回车打开选中的书即进入阅读。

![本地选书](../../../assets/guides/local-select.gif)

<Steps>

1. 在文件树里用 `↑/↓`(或 `j/k`)移动,回车展开/折叠目录。

2. 选中一本 `.txt`,回车进入阅读界面(左侧目录 + 右侧正文)。自动识别 UTF-8 / GBK 编码,并按「第…章」切分章节。

</Steps>

## 指定扫描目录

不想切到书库目录再启动?在本地小说页按 `s` 聚焦顶部路径框,输入任意目录路径后回车,即时切换扫描范围(该路径会存为历史,方便下次快速访问)。

![按路径扫描](../../../assets/guides/local-path.gif)

也可以在启动时用 `-l` 直接指定:

```bash
trn -l /path/to/your/books     # 扫描指定目录
trn -q                         # 跳过选择,直接续读上次的位置
```

<Aside type="note">本地小说按**字节偏移**记录章节位置以加速翻页;若文件在两次阅读之间被改动,旧的偏移可能失效,重新打开即可。</Aside>
