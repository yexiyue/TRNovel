听书功能基于 [Kokoro TTS](https://github.com/mzdk100/kokoro) 模型把正文转成语音边读边听,并在朗读时**按句高亮**当前位置。首次使用需要下载模型。

## 听书设置

在阅读界面切换到**阅读模式**(打开小说后按 `Tab`),再按 `T` 打开听书设置:

![听书设置](../../../assets/guides/tts-settings.gif)

设置项:

| 选项 | 说明 |
|------|------|
| 检查点模型 | TTS 模型文件状态;未下载时回车开始下载 |
| 语音数据 | 语音音色数据状态;未下载时回车开始下载 |
| 选择声音 | `←/→` 切换音色(如 `bf_vale` / `zf_001` …) |
| 播放速度 | `←/→` 调节倍速 |
| 音量 | `←/→` 调节音量 |
| 自动播放 | 进入阅读自动开始,读完自动跳下一章 |

操作:`↑/↓` 选择项,回车确认 / 开始下载,`←/→` 调节,`Esc` 取消下载。

## 安装模型

<Tabs>
  <TabItem label="自动下载(推荐)">
    在听书设置里分别选中「检查点模型」「语音数据」回车,即自动下载(支持断点续传)。两项都显示已下载即安装成功。

    <Aside type="note">模型较大(数百 MB),下载与播放需要可用音频设备。完整的下载 + 播放高亮演示见仓库 `docs/tapes/tts-play.tape`(真机录制)。</Aside>
  </TabItem>
  <TabItem label="手动下载">
    <Steps>
    1. 下载模型:[kokoro-v1.1-zh.onnx](https://github.com/mzdk100/kokoro/releases/download/V1.1/kokoro-v1.1-zh.onnx)
    2. 下载语音数据:[voices-v1.1-zh.bin](https://github.com/mzdk100/kokoro/releases/download/V1.1/voices-v1.1-zh.bin)
    3. 放到模型目录 `$HOME/.novel-tts/kokoro`:
        <Tabs>
            <TabItem label="macOS">`/Users/{用户名}/.novel-tts/kokoro`</TabItem>
            <TabItem label="Linux">`/home/{用户名}/.novel-tts/kokoro`</TabItem>
            <TabItem label="Windows">`C:\Users\{用户名}\.novel-tts\kokoro`</TabItem>
        </Tabs>
    4. 打开听书设置,确认两项均显示已下载。
    </Steps>
  </TabItem>
</Tabs>

## 开始听书

模型就绪后,在阅读模式按 `P` 播放 / 暂停。朗读时正文会按句高亮跟随,`+` / `-` 调整音量。

<Aside type="note">致谢 [mzdk100](https://github.com/mzdk100) 提供的 [`kokoro-tts`](https://github.com/mzdk100/kokoro) crate 与 Kokoro 模型。</Aside>
