## 一键安装脚本（推荐）

无需配置开发环境，复制对应平台的命令即可自动下载并安装最新版本：

<Tabs>
    <TabItem value="macos-linux" label="macOS / Linux">
        ```bash title="使用 Shell 脚本安装"
        curl --proto '=https' --tlsv1.2 -LsSf https://github.com/yexiyue/TRNovel/releases/latest/download/trnovel-installer.sh | sh
        ```
    </TabItem>
    <TabItem value="windows" label="Windows">
        ```powershell title="使用 PowerShell 安装"
        powershell -ExecutionPolicy Bypass -c "irm https://github.com/yexiyue/TRNovel/releases/latest/download/trnovel-installer.ps1 | iex"
        ```
    </TabItem>
    <TabItem value="homebrew" label="Homebrew">
        ```bash title="使用 Homebrew 安装（macOS / Linux）"
        brew install yexiyue/tap/trnovel
        ```
    </TabItem>
</Tabs>

安装脚本会自动检测平台、下载对应二进制并加入 PATH 环境变量。

## 包管理器安装

如果您已安装 Rust 或 Node.js 环境，可通过以下命令快速安装 TRNovel：

<Tabs>
    <TabItem value="cargo" label="cargo" icon="seti:rust">
        ```bash title="使用 Cargo 安装"
        cargo install trnovel
        ```
    </TabItem>
    <TabItem value="pnpm" label="pnpm" icon="pnpm">
        ```bash title="使用 pnpm 安装"
        pnpm add -g @trnovel/trnovel
        ```
    </TabItem>
    <TabItem value="npm" label="npm" icon="seti:npm">
        ```bash title="使用 npm 安装"
        npm i -g @trnovel/trnovel
        ```
    </TabItem>
</Tabs>

安装完成后，运行以下命令验证安装是否成功：

```bash
trnovel -h
```

## 手动下载二进制文件

也可前往 [GitHub Releases](https://github.com/yexiyue/TRNovel/releases/latest) 页面，根据平台下载预编译的压缩包（`.tar.gz` / `.zip`），解压后将可执行文件添加到系统 PATH 环境变量中。

:::caution[注意]
手动下载的用户需确保将可执行文件正确添加到 PATH 环境变量中，否则将无法在命令行中直接使用 trnovel 命令。
:::
