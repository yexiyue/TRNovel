## 包管理器安装

如果您已安装 Rust 或 Node.js 环境，可通过以下命令快速安装 TRNovel：

<Tabs>
    <TabItem value="cargo" label="cargo" icon="seti:rust">
        ```bash title="使用 Cargo 安装（推荐）"
        cargo install trnovel
        ```
    </TabItem>
    <TabItem value="pnpm" label="pnpm" icon="pnpm"> 
        ```bash title="使用 pnpm 安装"
        pnpm add -g @trnovel/trnovel
        ```
    </TabItem>
    <TabItem value="npm" label="npm" icon="seti:npm">
        ```bash title="使用 npm 安装"
        npm i -g @trnovel/trnovel
        ```
    </TabItem>
</Tabs>

安装完成后，运行以下命令验证安装是否成功：

```bash
trnovel -h
```

下载完成后，请解压文件并将可执行文件添加到系统 PATH 环境变量中。

:::caution[注意]
使用二进制文件的用户需确保将可执行文件正确添加到 PATH 环境变量中，否则将无法在命令行中直接使用 trnovel 命令。
:::
