## 如何退出 TRNovel？

您可以通过按 `q` 键退出 TRNovel 应用程序。或者，您也可以使用常见的终端退出命令，如 `Ctrl + C`。

## 如何前进或后退路由？

在 TRNovel 中，您可以使用以下快捷键进行路由导航：

- 前进路由：按 `G` 键
- 后退路由：按 `B` 键

## 如何查看快捷键列表？

您可以通过按 `I` 键查看当前页面的快捷键列表。
